# blinkit-sales-dashboard
A Power BI dashboard project analyzing Blinkit sales performance using Excel data.  This project explores item sales, outlet performance, fat content trends, outlet types, and customer preferences to deliver business insights and decision support.


## Blinkit Sales Dashboard (Power BI)

This project provides a comprehensive business intelligence dashboard analyzing Blinkit's sales and product performance. 
The dataset was cleaned and prepared using Excel, and the visualizations were built using Power BI to generate 
actionable retail insights.

### Key Features:
- Total sales and average sales breakdown
- Item category performance insights
- Fat content vs. sales analysis
- Outlet size and establishment year trends
- Outlet type performance comparison
- Average customer rating trends

### Tools Used:
- Power BI (Dashboard & Visualizations)
- Microsoft Excel (Data Cleaning & Preparation)

### Insights:
- Regular fat items contribute higher revenue compared to low-fat items.
- Tier 3 outlets show the highest sales performance.
- Supermarket Type 1 contributes maximum total revenue among all outlet types.
